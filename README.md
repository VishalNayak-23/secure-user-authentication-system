📌 Secure User Authentication System

A modern full-stack authentication system built with Node.js, Express, MongoDB, and React that provides a secure and scalable way to manage user accounts. The application allows users to register, log in, and access protected routes using JWT-based authentication and HTTPOnly cookies for enhanced security.

✨ Key Features

🔐 Secure user registration and login system
🔑 JWT-based authentication for protected routes
🛡 Password hashing using bcrypt for enhanced security
🍪 HTTPOnly cookies for secure session handling
👤 Role-based access control support
⚡ Fast frontend built with React and Vite
🎨 Modern responsive UI using TailwindCSS
🔎 Input validation using express-validator
🌐 RESTful API communication with Axios
🔒 Security best practices using Helmet and CORS

🛠 Tech Stack

Backend (JavaScript / Node.js)
- Express.js - Web framework
- MongoDB + Mongoose - Database & ODM
- jsonwebtoken - JWT authentication
- bcryptjs - Password hashing
- Helmet - Security headers
- CORS - Cross-origin requests
- cookie-parser - Cookie handling
- express-validator - Input validation
- dotenv - Environment variables

Frontend (JavaScript / JSX)
- React 18 - UI library
- Vite - Build tool
- React Router v6 - Client-side routing
- Axios - HTTP requests
- TailwindCSS - Utility-first CSS
- Context API - Global state management
